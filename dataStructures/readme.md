# Data Structures golang

## Arrays vs Linked List  

| Action        | Array   | Linked  |
| ------------- |:-------:| -------:|
| Access        | O(1)    |  O(n)   |
| Search        | O(n)    |  O(n)   |
| Prepend       | O(1)t   |  O(1)   |
| Append        | O(n)    |  O(1)   |
| Delete        | O(n)    |  O(n)   |
